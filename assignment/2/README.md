## 단축url 생성앱 (네이버 개발자 센터 API 이용)

#### 긴 url을 상단 editText에 입력하고 'url 단축하기' 버튼을 터치하면 단축된 url을 받아와 디바이스 클립보드에 복사.

네이버 단축 URL API 가이드 > https://developers.naver.com/docs/utils/shortenurl/
